数据集与处理代码

    ./MRBrainS_raw：原始数据集，用nii格式打包的

		MRBrainS_rawdata2matpng.m：处理nii格式的原始数据，转换成.mat文件和png图片
		MRBrainS_mat2tfrecord_enlarge.py：将.mat格式数据转换成tfrecord，其中包含数据增扩，这里需要了解tfrecord数据格式，以及数据增扩都包含哪些操作

	./data/MRbrain_mat_training：mat格式训练集
	./data/MRbrain_mat_testing：mat格式测试集
	./data/tfrecords：tfrecord格式训练集和验证集，需要了解训练集、测试集、验证集是如何划分的，各自的作用

代码（基于tensorflow深度学习框架）

	./MRBrains_enlarge_seg.py：主要代码文件，training和testing都在这里面，里面涉及到较多参数设置，需要一一理解
	./ops.py：主文件中用到的函数都在这里，包括卷积层、池化层、反卷积层等

目录说明
	
	./data：存放代码使用的数据
	./logs：模型训练过程中保存的日志文件，可用于查看loss曲线
	./models：存放训练过程中生成的模型文件
	./samples：存放训练过程中生成的验证数据预测结果
	./test：存放测试生成的结果
